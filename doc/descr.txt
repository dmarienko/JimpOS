Краткое техническое описание JimpOS
-----------------------------------

   Цель проекта JimpOS - реализация многозадачной, платформенно не зависимой, встраиваемой 
операционной системы. Архитектура имеет 3-х слойную реализацию - платформенно-зависимая часть, 
интерпретатор java-байт кода и платформенно-независимая. 

   Зависимый от используемой платформы первый слой (x86) представляет собой микроядро, написанное 
на языках С и Assemler. Оно обеспечивает начальную загрузку, управление прерываниями, 
многопоточность, реализует механизм выделения/освобождения памяти. Дает пользователям возможность 
работы с несколькими приложениями. Объем кода достаточно мал и не превышает 12 Кбайт.

   Следующий слой - интерпретатор java байт кода (JimpVM). Отвечает за интерпретацию java байт 
кода и обеспечивает интерфейс между ресурсами машины (hardware) и программами написанными на Java.

   Третий, независимый от платформы слой, реализован на языке Java и представляет собой 
абстрактную модель ресурсов машины на которой выполняется. Нити в java приложениях реализованы 
как отдельные процессы, что позволяет добиться высокой производительности в многопоточных 
приложениях.

 Использованные языки программирования - Java, C, Assembler
 Платформа разработки - Linux(2.4.18), gcc-2.95.3(2.96), jikes-1.15
